# NAME

Conch::DB::ResultSet::Build

# DESCRIPTION

Interface to queries involving builds.

# METHODS

## user\_has\_role

Checks that the provided user\_id has (at least) the specified role in at least one build in the
resultset.

Returns a boolean.
