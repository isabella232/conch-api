# NAME

Conch::DB::ResultSet::UserWorkspaceRole

# DESCRIPTION

Interface to queries involving user/workspace roles.

# METHODS

## with\_role

Constrains the resultset to those user\_workspace\_role rows that grants (at least) the specified
role.
