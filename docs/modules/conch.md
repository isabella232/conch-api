# NAME

Conch - Setup and helpers for Conch Mojo app

# SYNOPSIS

```
Mojolicious::Commands->start_app('Conch');
```

# METHODS

## startup

Used by Mojo in the startup process. Loads the config file and sets up the
helpers, routes and everything else.
