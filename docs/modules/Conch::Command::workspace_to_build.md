# NAME

workspace\_to\_build - convert workspace content to a build

# SYNOPSIS

```
bin/conch workspace_to_build [long options...] <workspace name> [workspace name] ...

    -n --dry-run  dry-run (no changes are made)

    --help        print usage message and exit
```
