# clean\_roles - clean up unnecessary user\_workspace\_role entries

## SYNOPSIS

```
bin/conch clean_roles [-nv] [long options...]
    -n --dry-run  dry-run (no changes are made)
    -v --verbose  verbose

    --help        print usage message and exit
```
